# CredPro Redesign – Customer Workflow (Frontend)

This project showcases the redesign of the **Customer Editor Workflow journey in CredPro**, focusing on improving usability, structure, and visual clarity across key RM-facing screens.

The implementation is built using **HTML and CSS**, representing UI/UX improvements across the customer onboarding and workflow experience.

---

## 🎯 Objective

- Redesign the customer onboarding journey  
- Improve screen structure and information hierarchy  
- Enhance usability for Relationship Managers (RMs)  
- Create a cleaner, more intuitive workflow experience  

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- Flexbox / Layout styling  
- Static UI implementation (no backend integration)  

---

## ▶️ How to Run

1. Download or clone the repository  
2. Open `index.html` in your browser
